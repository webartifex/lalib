=========================================

This is the documentation for the `lalib`_ package
whose goal it is to serve as a resource
for software engineers and data scientists
to learn about linear algebra via reading and writing code.


Prerequisites
-------------

`Python`_ 3.7+ is needed.
The package depends only on core Python (incl. the standard library).


Installation
------------

`lalib`_ is available on `PyPI`_ via `pip`_:

.. code-block:: console

   $ pip install lalib

It is recommended to install the package into a `virtual environment`_.
